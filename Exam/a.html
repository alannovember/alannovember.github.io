Question 2 Answer:

Css stands for cascading sheet styles. It is used to define the way a web page is set up (i.e.- font styles, background pictures, size of elements.) W3C or World Wide Web Consortium maintains CSS. We are using CSS3 for our pages. There are 3 ways to add css to a web page. They are: Inline style, Embedded Style, and External Style.

Question 3 Answer:

The three DTDs are Transitional, Frameset, and Strict. Transitional is used for older documents that contain depricated features. Frameset is used for pages containing frames. Strict is used for documents that only contain the latest standards. It does not support presentational features, frames, inline frames, or depricated elements.
